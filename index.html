WHAT IS ORACLE · 关于本系统
命
不是算命
ORACLE 不预测你的未来，也不给你贴标签。它用八字作为入口，帮你看清自己天生的认知结构——你怎么处理信息、做决定、消耗能量。
识
认知操作系统
每个人出生时自带一套认知操作系统。八字记录了这套系统的初始参数：优势模块、盲区、能量分配方式。ORACLE 把它翻译成现代语言。
还
还原，不改变
你听过太多"你应该变成什么样"。ORACLE 只问一件事：你本来是什么样的？所有输出都指向还原，而不是改造。
HOW TO USE · 使用步骤
STEP 01
配置 API Key
填入 DeepSeek 或 OpenAI 的 API Key，保存到浏览器本地。Key 不经过任何服务器。
STEP 02
输入四柱八字
点击年月日时各柱的天干地支按钮，从弹出菜单中选字。不知道八字可在网上搜索"生辰八字查询"。
STEP 03
填写基本信息
填入姓名（可选）、性别、当前最关注的方向，让分析更有针对性。
STEP 04
启动分析 & 追问
点击"启动分析"，ORACLE 流式返回五维认知图谱。结果出来后可继续追问。

天干 · Heavenly Stems
甲乙丙丁戊己庚辛壬癸，共十个字，代表能量的性质与方向——木的生发、火的扩散、土的稳固、金的收敛、水的渗透。
地支 · Earthly Branches
子丑寅卯辰巳午未申酉戌亥，共十二个字，对应十二月令与时辰，记录能量落地的方式与节律。
四柱 · Four Pillars
年柱（出生年）、月柱（出生月）、日柱（出生日）、时柱（出生时辰）。每柱一天干一地支，合计八字。
日主 · Day Master
日柱天干是"日主"，代表你这个人的核心能量属性，是八字分析的锚点。ORACLE 以日主为中心展开认知解读。
不知道自己的八字？可在搜索引擎搜索「生辰八字在线排盘」，输入出生年月日时即可获得四柱。
